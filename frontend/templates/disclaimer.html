Sobre o Software

O LabGas Manager (v2.9.0) é um sistema web de código aberto desenvolvido para gestão e rastreabilidade de dados em laboratórios químicos acadêmicos. Construído com Flask, Jinja2 e Supabase (PostgreSQL), o sistema permite o registro e acompanhamento de cilindros de gás, elementos analisados, leituras, amostras e medições de pressão, com dashboard analítico e controle de acesso baseado em papéis (RBAC).

Este software foi desenvolvido no âmbito acadêmico como ferramenta de apoio à gestão laboratorial, alinhado aos princípios das Boas Práticas de Laboratório (BPL).
